Usability test Jibbe

In de eerste connection les moesten we een soort wireframe maken van ons portfolio.

Tijdens deze les vroeg Witek of ik een Usability test wilde doen bij iemand uit de klas. Ik heb dit toen gedemonstreerd aan de klas met behulp van het design van Jibbe.

Omdat wij over het algemeen al verschillende pagina's hebben gemaakt, kun je de navigatie al helemaal simuleren. In het geval van Jibbe had hij alleen nog een homepage. Hierdoor was een echte Usability test moeilijker. Daarom heb ik verteld over de goede punten van het design en de dingen waar hij nog op moest letten.

Ik heb uitgelegd dat je altijd moet kijken hoe je gaat navigeren. Het is namelijk van belang dat het zo duidelijk mogelijk is en dat de gebruiker zo snel mogelijk bij zijn/haar bestemming komt.

Daarnaast heb ik een aantal designtips gegeven, zoals: - Plaats je logo links boven in het beeld, want daar kijken mensen het eerst naar.
- Gebruik zo veel mogelijk afbeeldingen, videoâ€™s en illustraties.
- Probeer het aantal kleuren te beperken en niet te veel opvallende kleuren te gebruiken.
- Kijk wie je doelgroep is en pas je design daarop aan.

Inspiratie opdoen

Voordat ik ben begonnen met het ontwerpen van mijn portfolio, heb ik inspiratie opgedaan. Dit kan op verschillende manieren, zoals kijken naar afbeeldingen, navigatie mogelijkheden, websites met uitstraling die je aanstaat eb kleurencombinaties die goed bij elkaar passen. Als je op al die losse onderdelen inspiratie opdoet, kun je uiteindelijk tot een complete unieke website komen.

Maar hoe verzamel je inspiratie? Dat kan op heel veel verschillende manieren, maar ik gebruik over het algemeen een Pinterest-bord. Aan het begin van het project heb ik dan ook een bord voor mijn portfolio gemaakt. Deze heb ik weer opgedeeld in de selecties die hieronder te zien.



Inspiratie opdoen kun je niet plannen. Het ene moment heb je veel inspiratie en het andere moment heel weinig. Daarom is het belangrijk dat je het opschrijft of ergens bewaart wanneer je inspiratie hebt. Door het Pinterest-bord kon ik altijd een idee opslaan. Als ik even tijd voor mezelf had, scrolde ik door Pinterest en alles wat me interessant leek, sloeg ik op. En na een paar weken had ik al een behoorlijk gevuld Pinterest-bord. Op deze manier kreeg ik een beter beeld over welke kant ik op wilde gaan met mijn design.

Onderzoek navigaties

Omdat er op deze website heel veel informatie moet worden verwerkt, is het van belang dat de gebruiker alles makkelijk kan vinden. De juiste navigatie is dan dus ook cruciaal. Ik heb heel veel verschillende navigatiemogelijkheden bekeken om te ontdekken welke zou(den) kunnen werken voor mijn portfolio.

Een van de websites waar ik veel op heb gekeken, is die van awwwards.com. Hier staat een selectie van websites met interessante navigatie, bijvoorbeeld de website van Damian Kujawa. Deze heeft een hele bijzondere slideshow als navigatie. Ik had hier dan ook het idee bij gekregen om in mijn portfolio elk vak een afbeelding te maken.

Na heel veel zoeken en uitproberen, ben ik toch gegaan voor een hamburger menu. Dit kwam vooral doordat ik de andere navigatie niet werkend kreeg. Ik heb er meerdere mensen naar laten kijken, maar we konden de fout(en) niet vinden.

Een hamburger menu is ook overzichtelijk, dus dat was ook een goede optie voor mijn portfolio.

Logo design

Een logo is een identificatie van je merk, geen uitleg ervan. Je moet een logo maken dat mensen gaan herkennen als jouw merk.

Eerst heb ik opgeschreven welke woorden er bij mij naar boven kwamen als ik dacht aan Smart Industry. Nadat ik dit had opgeschreven, ben ik illustraties gaan zoeken die horen bij deze woorden. Deze heb ik geplaatst in mijn Pinterest-bord. Zo had ik een overzicht van ideeÃ«n voor mijn logo. Ik ben toen een aantal opties gaan schetsen op papier. Deze heb ik voorgelegd aan verschillende mensen met de vraag: â€˜Welke vind jij het meest passen bij Smart Industry?â€™

Daaruit heb ik een variant gekozen en deze ben ik in Illustrator gaan uitwerken. Toen ik het logo had staan, heb ik er zowel een zwarte als een witte versie van gemaakt. Dit heb ik gedaan zodat ik het witte logo kan gebruiken bij een donkere achtergrond en vice versa. Aangezien ik de titel van het vak bij het logo wilde verwerken, heb ik hierbij ook naar verschillende opties gekeken. Uiteindelijk heb ik daar een van gekozen en deze is terug te vinden bovenaan elke pagina.

Sitemap

Omdat er op het portfolio heel veel informatie moet komen was het belangrijk dat de navigatie duidelijk was. Daarom heb ik voor mezelf een sitemap gemaakt.
